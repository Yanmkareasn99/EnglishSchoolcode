English School 管理システム

このリポジトリは、簡易的な英会話スクール向け管理システムの Java 実装です。CSV ファイルでデータを読み書きし、講師、受講生、レッスン、料金に関する基本的な操作を提供します。

**主な機能**
- 生徒（`Student`）の読み込み・一覧表示
- 教師（`Teacher`）の読み込み・一覧表示
- レッスン（`Lesson` / `LessonCost`）の管理と料金計算
- CSV 入出力ユーティリティ（`CsvUtil`）によるデータの永続化（ファイルベース）
- 役割別メニュー（`StaffMenu` / `StudentMenu` / `TeacherMenu`）を通した操作

**既知の問題 / 未実装の点**
- 入力バリデーションが限定的で、想定外の入力で例外が発生する可能性があります。
- 同時実行（複数ユーザーによる同時更新）を想定したロックやトランザクションは未実装です。
- 単体テストや自動テストが整備されていません。
- 一部のビジネスロジックはメニュークラスに混在しており、責務分離が不十分です（リファクタ対象）。
- エラーメッセージやログ出力が限定的で、デバッグしづらい箇所があります。

**実行方法（簡易）**
Windows のコマンドプロンプト / PowerShell でプロジェクトルートに移動して以下を実行します。

```bash
javac -d out src/*.java
java -cp out EnglishSchool
```

（上記は簡易手順です。IDE（Eclipse/IntelliJ）でプロジェクトを開いて実行することを推奨します。）
