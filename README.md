英会話管理システム

このリポジトリは、簡単な英会話スクール向け管理システムです。

**主な機能**
- 生徒（`Student`）
生徒IDを入力して
　生徒情報確認、レッスン予約、レッスン確認　ができます。

- 教師（`Teacher`）
教師生徒IDを入力して
　


- 受付スタッフ（`Staff`）


- CSV 入出力（`CsvUtil`）によるデータ管理と保存（ファイルベース）
- 役割別メニュー（`StaffMenu` / `StudentMenu` / `TeacherMenu`）を通した操作

**既知の問題 / 未実装の点**
- 一部の　Scanner で Expection の可能性があります。
- クレジットカードのトランザクションは未実装です。。
- 一部のビジネスロジックはメニュークラスに混在しており、責務分離が不十分です（リファクタ対象）。
- エラーメッセージやログ出力が限定的で、デバッグしづらい箇所があります。
