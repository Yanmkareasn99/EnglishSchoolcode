　英会話スクール管理システム

このリポジトリは、簡易的な英会話スクール向け管理システムです。

**主な機能**
- Main Menu
　 生徒、講師、受付から自分の立場を選択します。

- 生徒（`Student`）
　 生徒IDを入力してログインします。
　 ログイン後、生徒情報確認、レッスン予約、レッスン確認　ができます。

- 講師（`Teacher`）
　 講師IDを入力してログインします。
　 ログイン後、レッスン一覧、出席登録　ができます。


- 受付スタッフ（`Staff`）
　 生徒登録、生徒一覧、生徒情報変更、ポイント購入、レッスン予約、レッスン確認、
　 レッスンキャンセル、レッスン単価、講師登録、講師一覧、生徒退会、売り上げ確認　ができます。

- CSV 入出力（`CsvUtil`）によるデータ管理と保存（ファイルベース）
- 役割別メニュー（`StaffMenu` / `StudentMenu` / `TeacherMenu`）を通した操作

**既知の問題 / 未実装の点**
- 一部の　Scanner で Exception の可能性があります。
- エラーメッセージやログ出力が限定的で、デバッグしづらい箇所があります。
